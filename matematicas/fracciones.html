Fracciones y número mixto

Numerador y denominador, propias e impropias, número mixto y comparación.

¿Qué es una fracción?

Una fracción representa una o varias partes iguales de un todo. Se escribe con dos números separados por una raya:

NumeradorDenominador
Cuántas partes se toman (arriba)En cuántas partes se divide el todo (abajo)

Cómo se leen las fracciones

Ejemplo: 3/5 se lee "tres quintos".

Fracciones propias e impropias

Fracción propia

El numerador es menor que el denominador. Representa menos de la unidad.

Fracción impropia

El numerador es mayor o igual que el denominador. Representa una unidad completa o más.

El número mixto

Una fracción impropia se puede escribir como un número mixto: un número entero más una fracción propia.

Cómo convertir: divide el numerador entre el denominador. El cociente es el número entero; el resto es el nuevo numerador, con el mismo denominador.
Ejemplo: 7 ÷ 3 = 2 y sobran 1 → 7/3 = 2 1⁄3

Comparar fracciones

Mismo denominador

Es mayor la que tiene el numerador más grande.

Ejemplo: 4/5 > 3/5

Mismo numerador

Es mayor la que tiene el denominador más pequeño (las partes son más grandes).

Ejemplo: 2/3 > 2/5
