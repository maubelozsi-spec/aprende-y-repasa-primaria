Números decimales

Relación con las fracciones, comparación, suma y resta.

¿Qué es un número decimal?

Un número decimal tiene una parte entera y una parte decimal, separadas por una coma.

Unidades,DécimasCentésimasMilésimas
3,457
3,457 se lee "tres unidades cuatro décimas, cinco centésimas y siete milésimas".

Relación con las fracciones

Décimas

Una cifra decimal: 0,3 = 3/10

Centésimas

Dos cifras decimales: 0,25 = 25/100

Milésimas

Tres cifras decimales: 0,125 = 125/1000

Comparar y ordenar decimales

Cómo comparar

Primero se compara la parte entera. Si es igual, se comparan las décimas; si también son iguales, las centésimas, y así sucesivamente.

Ejemplo: 3,45 < 3,5 (las décimas: 4 < 5) · 2,7 = 2,70 (los ceros a la derecha no cambian el valor)

Suma y resta de decimales

Cómo se suman o restan

Se colocan los números en columna alineando las comas (y por tanto las unidades con unidades, décimas con décimas...), y se suma o resta como con números naturales.

Ejemplo: 2,5 + 1,3 = 3,8

Redondear números decimales

La misma regla del redondeo

Para aproximar un decimal a las décimas, centésimas..., se mira la cifra que está justo a la derecha de esa posición: si es 5 o más, la cifra sube una unidad; si es menor que 5, se queda igual. Todas las cifras posteriores se eliminan.
