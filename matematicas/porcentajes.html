Porcentajes

Relación con los decimales y cálculo del porcentaje de un número.

¿Qué es un porcentaje?

Un porcentaje indica una cantidad "de cada 100". Se escribe con el símbolo %.

Ejemplo: 25% significa "25 de cada 100".

Relación entre decimal y porcentaje

De porcentaje a decimal

Se divide entre 100 (la coma se mueve dos lugares a la izquierda).

Ejemplo: 25% = 0,25

De decimal a porcentaje

Se multiplica por 100 (la coma se mueve dos lugares a la derecha).

Ejemplo: 0,4 = 40%

Calcular el porcentaje de un número

Cómo se calcula

Se multiplica el número por el porcentaje y se divide entre 100.

Ejemplo: 20% de 50 → 50 × 20 ÷ 100 = 1000 ÷ 100 = 10.
Truco: para el 10% de un número, basta con dividir entre 10. Para el 50%, dividir entre 2. Para el 25%, dividir entre 4.
