Siglas y abreviaturas

Dos formas distintas de acortar palabras y expresiones.

Las siglas

¿Qué son?

Se forman con las letras iniciales de varias palabras, y se pronuncian letra por letra o como una palabra nueva. Se escriben en mayúsculas y sin puntos.

Ejemplos: ONU (Organización de las Naciones Unidas), ADN (ácido desoxirribonucleico), OVNI (objeto volador no identificado)

Acrónimo: cuando una sigla se pronuncia como una palabra normal (no letra por letra), como "OVNI" o "láser", se llama acrónimo.

Las abreviaturas

¿Qué son?

Son la reducción de una palabra o expresión, eliminando letras finales o centrales, y terminan casi siempre con un punto.

Ejemplos: Sr. (señor), pág. (página), etc. (etcétera), Dr. (doctor), c/ (calle)

Diferencias entre sigla y abreviatura

SiglaAbreviatura
Se forma conletras iniciales de varias palabrasreducción de una sola palabra

Truco: si ves varias letras mayúsculas juntas representando varias palabras, es una sigla. Si ves una palabra recortada terminada en punto, es una abreviatura.
